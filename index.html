Welcome to LianjaX

LianjaX is an Open Source Community for sharing Libraries, Tools, and Templates created for use with Lianja.

The Libraries, Tools and Templates made available here are all open source licensed under the Apache 2 OSS license. Any Lianja developer can join and become a member of LianjaX, help and propose new LianjaX projects, and generally work towards a common goal of building a collection of useful open source technologies to help others become more productive in Lianja.

Getting Started

The built-in GIT instructions for getting started, at http://help.github.com/ are very good and easy to follow. Once Git is installed, Scott Chacon's Pro Git is a great book, and is available online at http://git-scm.com/book.
